# 纯碱期货分析看板 - 设计理念

## 选定设计方案：**数据驱动的专业金融仪表板**

### 设计哲学
打造一个**严谨、专业、信息密度高**的金融数据分析平台。设计理念强调**清晰的信息层级、精准的数据展示、以及冷静的视觉语言**，让用户能够快速获取关键信息并做出决策。

### 核心原则

1. **信息优先**：所有设计决策都围绕数据的清晰展示和易读性展开，避免过度装饰。
2. **专业克制**：采用深色背景 + 高对比度文字，营造严肃、可信的专业氛围。
3. **功能美学**：每一个视觉元素都有其功能性，不存在纯装饰性的设计。
4. **数据可视化优先**：图表、指标卡、趋势线是核心视觉元素，应该占据最显眼的位置。

### 色彩哲学

**主色板**：
- **背景**：深灰蓝 (`#0f172a`) - 深沉、专业、减少眼睛疲劳
- **主色**：蓝绿色 (`#06b6d4` / Cyan-500) - 代表数据、科技、信任
- **强调色**：琥珀色 (`#f59e0b`) - 用于警示、上升趋势、重要数据
- **文字**：白色 (`#ffffff`) 和浅灰 (`#e5e7eb`) - 高对比度，易读性强
- **边框/分割线**：深灰 (`#1e293b`) - 微妙的结构感

**情感意图**：
- 深色背景传达**专业、严肃、可信**的品牌形象
- 青色强调色代表**数据、科技、动态**
- 琥珀色用于**突出重要趋势和警示信息**

### 布局范式

采用**非对称的侧边栏 + 主内容区**布局：
- **顶部导航栏**：品牌名称、快速导航、主题切换
- **左侧侧边栏**（可折叠）：导航菜单、快速筛选器
- **主内容区**：分为多个**卡片网格**，每个卡片展示一个关键指标或图表
- **响应式设计**：移动设备上侧边栏隐藏，卡片堆叠排列

### 签名元素

1. **数据卡片**：带有细微阴影、圆角、半透明背景的卡片，用于展示关键指标
2. **渐变分割线**：从青色到透明的渐变线，用于分隔不同的内容区域
3. **微交互**：卡片悬停时轻微上浮、边框发光效果
4. **数据图表**：使用 Recharts，采用青色和琥珀色作为主要图表颜色

### 交互哲学

- **悬停反馈**：卡片、按钮在悬停时出现**微妙的上浮效果**和**边框发光**
- **加载动画**：使用脉冲效果或渐进式加载，传达**数据正在更新**的感觉
- **数据更新**：新数据到达时，相关卡片出现**轻微的闪烁或高亮**效果
- **页面过渡**：路由切换时使用淡入淡出效果，保持视觉连贯性

### 动画指南

- **卡片进入**：从下方淡入 + 轻微上升（`translateY` + `opacity`），持续 400ms
- **悬停效果**：卡片上浮 4px，阴影增强，边框发光（青色 glow），持续 200ms
- **数据更新**：目标卡片出现 1 次脉冲效果（背景闪烁），持续 600ms
- **按钮交互**：点击时出现涟漪效果，背景颜色变化，持续 150ms

### 排版系统

**字体选择**：
- **标题字体**：`Poppins` (Google Fonts) - 现代、清晰、数字感强
- **正文字体**：`Inter` - 高可读性、专业、适合数据展示

**层级规则**：
- **H1 (页面标题)**：`Poppins`, 32px, 700 weight, 青色
- **H2 (区域标题)**：`Poppins`, 24px, 600 weight, 白色
- **H3 (卡片标题)**：`Poppins`, 16px, 600 weight, 浅灰
- **正文 (Body)**：`Inter`, 14px, 400 weight, 浅灰
- **数据值**：`Poppins`, 28px, 700 weight, 白色（用于大数字展示）
- **标签/说明**：`Inter`, 12px, 500 weight, 深灰（辅助文字）

### 设计一致性检查清单

- [ ] 所有卡片使用相同的圆角半径和阴影样式
- [ ] 所有文字颜色要么是白色，要么是浅灰，确保对深色背景的对比度 > 7:1
- [ ] 所有交互元素（按钮、链接）在悬停时都有明确的视觉反馈
- [ ] 图表颜色只使用青色和琥珀色两种主色
- [ ] 页面间的过渡动画保持一致性（都是淡入淡出）
- [ ] 响应式设计在移动、平板、桌面三个断点都经过测试
